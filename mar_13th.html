Visualization:Basic bar chart with multiple series

The main animation in this visualization is that when one hovers over the bars a text box pops up with the name of the City and its population size.
The best way to accomplish this using d3 is by calling on() with mouseover (it's equivalent to CSS's hover).

      svg.selectAll('rect')
        .data(dataset)
        .enter()
        .append("rect")//ommitting set attributes
        .on("mouseover",function()
        {
          d3.select(this)
            .attr("fill","orange")
            //code to creat text box with info
        })
